# 屏幕监控客户端

## 概述
基于 C# + WPF + .NET 6 开发的桌面监控客户端，负责屏幕截图采集、剪切板监控和区块链地址安全检测。

## 主要功能
- 定时屏幕截图（1 FPS）
- 剪切板实时监控
- 区块链地址检测（支持BTC、ETH、TRC20等）
- 安全告警高清截图
- 自动重连机制

## 技术特性
- .NET 6 原生性能
- WPF 现代化界面
- 异步任务处理
- 内存优化管理

## 开发环境
- Visual Studio 2022
- .NET 6.0 SDK
- Windows 10/11

## 构建说明
```bash
dotnet build
dotnet run
```
